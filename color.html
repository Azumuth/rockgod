Color



Primary colors

The RGD logo should be used in color whenever possible.

Logo standard colors should are:

  • #F7931E (orange)
  • #050505 (black)
  • #F9F8F7(white)

Alternative colors

Alternative colors for the logo may be used where applicable, as per the examples on the page. The following colors should be used:

  • #0071BC (blue)
  • #39B54A (green
  • #C1272D(red)

Formats

For screen display (web, email, ppt), use a .JPG or .PNG file. For print, use an .EPS file
